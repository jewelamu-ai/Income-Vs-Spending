Option 3: Browser-Based Icon Generation (No Python needed)

Open icon-generator-canvas.html in your browser, then:

  1. Click "Generate All Icons"
  2. Click "Download PNG" for each icon size
  3. Save them in the icons/ folder
  4. Then start the server:
    python -m http.server 8000

Option 3: Generate proper icons (optional)

npm install && npm run generate-icons

This creates PNG icons for better PWA compatibility.
